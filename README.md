## Documentación del Proyecto

### 1. ¿Qué es GitHub y qué relación tiene con Git?
GitHub es un portal y servicio de alojamiento en la nube para repositorios Git. Mientras que Git es la herramienta de línea de comandos que gestiona el control de versiones localmente en tu ordenador, GitHub facilita el almacenamiento remoto, la copia de seguridad y el trabajo colaborativo entre varios desarrolladores.

### 2. Tipos de repositorios en GitHub
Existen principalmente dos tipos:
- **Públicos:** Cualquiera puede ver el código, pero solo tú (y quien tú digas) puede editarlo.
- **Privados:** Solo tú y las personas que invites explícitamente podéis ver y editar el código.
